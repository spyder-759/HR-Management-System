# HR Management System

A comprehensive HR Management System built with PHP, MySQL, HTML, CSS, and JavaScript.

## Quick Setup

### 1. Database Setup
```sql
CREATE DATABASE hr_management_system;
-- Import database.sql file
```

### 2. Configure
Edit `includes/config.php` with your database credentials.

### 3. Access
- URL: `http://localhost/hrms/`
- Default Login: `admin` / `admin123`

## Features
- 🔐 Secure Authentication
- 📊 Interactive Dashboard
- 👨‍💼 Recruitment Management
- 🧑‍💻 Employee Management
- 📅 Leave & Attendance
- 📈 Performance Reviews
- 📱 Responsive Design

## Requirements
- PHP 7.4+
- MySQL 5.7+
- Apache/Nginx
- Modern Browser
